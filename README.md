# Tabla Completa de Medidas del Kilogramo 📊

Una herramienta web interactiva para convertir y comprender las diferentes unidades de masa basadas en el kilogramo.

## Características ✨

### 📋 Tabla Completa
- Tabla detallada con todas las unidades de masa
- Equivalencias en gramos y kilogramos
- Símbolos internacionales
- Diseño responsive y accesible

### 🧮 Calculadora Universal
- Convierte entre **cualquier par de unidades**
- Soporta: kg, hg, dag, g, dg, cg, mg
- Validación de entrada en tiempo real
- Formateo de números con separadores locales
- Soporte para Enter key

### 🚀 Conversiones Rápidas
- Botones predefinidos para conversiones comunes
- KG ↔ Gramos
- KG ↔ Miligramos
- Más opciones disponibles

### 🎨 Diseño Moderno
- Interfaz limpia y profesional
- Colores degradados atractivos
- Animaciones suaves
- Soporte para modo oscuro automático
- Completamente responsive (móvil, tablet, desktop)

## Tecnología 💻

- **HTML5**: Estructura semántica
- **CSS3**: Estilos modernos con Flexbox y animaciones
- **Vanilla JavaScript**: Sin dependencias externas

## Instalación 🚀

1. Clona este repositorio:
```bash
git clone https://github.com/dragostefan2012/Tabla-Completa-de-Medidas-del-Kilogramo.git
```

2. Abre `index.html` en tu navegador
   - Sin necesidad de servidor
   - Funciona offline

## Uso 📖

### Conversión Manual
1. Ingresa un valor en el campo "Valor a convertir"
2. Selecciona la unidad de origen
3. Selecciona la unidad de destino
4. Haz clic en "Convertir" o presiona Enter
5. Verás el resultado inmediatamente

### Conversiones Rápidas
Haz clic en cualquiera de los botones de conversión rápida con valores predefinidos.

## Unidades Soportadas 📏

| Unidad | Símbolo | Equivalencia |
|--------|---------|--------------|
| Kilogramo | kg | 1000 gramos |
| Hectogramo | hg | 100 gramos |
| Decagramo | dag | 10 gramos |
| Gramo | g | 1 gramo |
| Decigramo | dg | 0.1 gramos |
| Centigramo | cg | 0.01 gramos |
| Miligramo | mg | 0.001 gramos |

## Ejemplos de Uso 💡

### Ejemplo 1: Convertir kg a g
- Ingresa: `2.5`
- Desde: `Kilogramos (kg)`
- Hacia: `Gramos (g)`
- Resultado: `2.5 kg = 2,500 g`

### Ejemplo 2: Convertir mg a kg
- Ingresa: `500000`
- Desde: `Miligramos (mg)`
- Hacia: `Kilogramos (kg)`
- Resultado: `500,000 mg = 0.5 kg`
